### Why I built it

I wanted to build something that felt like a real product instead of another tutorial project. The goal wasn't just to integrate AI, but to understand everything that goes into building a complete application.

### Highlights

- Secure authentication using JWT
- Persistent chat history
- AI image generation
- Responsive UI
- Clean and intuitive user experience

### What I learned

This project strengthened my understanding of backend architecture, API design, authentication, database design, media handling, and building applications that feel complete instead of just functional.

### Next Improvements

- Streaming AI responses
- Better backend architecture
- Automated testing
- Performance optimization
